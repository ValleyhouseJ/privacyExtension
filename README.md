# Privacy Scanner Extension

En Chrome browser extension der scanner hjemmesider for cookies og giver brugeren indsigt i deres online privatliv.

## Installation

### 1. Download projektet

Klik på den grønne **Code**-knap øverst på denne side, og vælg **Download ZIP**.

Pak ZIP-filen ud et valgfrit sted på din computer.

### 2. Åbn Chrome Extensions

Åbn Google Chrome og gå til:

```
chrome://extensions
```

Eller: Klik på de tre prikker -> **Udvidelser** -> **Administrer udvidelser**

### 3. Aktivér Udviklertilstand

Slå **Udviklertilstand** til i øverste højre hjørne.

### 4. Indlæs udvidelsen

Klik på **Læs upakket** og vælg den mappe, du pakkede projektet ud til (mappen der indeholder `manifest.json`).

### 5. Færdig!

Udvidelsen er nu installeret. Du finder ikonet i Chrome's toolbar øverst til højre.

Klik på ikonet for at åbne Privacy Scanner på en vilkårlig hjemmeside.

## Brug

1. Gå til en hjemmeside
2. Klik på Privacy Scanner-ikonet i toolbaren
3. Klik på **Scan page**
4. Se resultatet med cookies opdelt i kategorier

## Krav

- Google Chrome
- Ingen yderligere installation påkrævet
